Midi Player | 11-04-2025

\[ \nabla f= \binom { \Delta x T \cdot \sin (\Delta x T) \cdot \sin (\Delta y T) \cdot \sin(t) \cdot s } { \Delta y T \cdot \cos(\Delta y T) \cdot \cos (\Delta x T) \cdot \cos(t) \cdot s } \]

This project is an online MIDI player. It accepts any MIDI file as input, plays the music using an oscillator and visualizes the sound.

By using a formula inspired by the Chladni plate experiment, we visualize sound waves generated by the MIDI file through interference patterns.

The applied force on each dot is derived from the wave gradient created by each note. The variables are:

  • Δx, Δy — distance between the dot and frequency source
  • T — period, i.e., \( T = \frac{1}{f} \) where \( f \) is frequency
  • t — current time in the song
  • s — speed factor

Either use the default example and just press PLAY! or upload your own MIDI file.
